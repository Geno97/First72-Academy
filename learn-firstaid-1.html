About this course

Awareness content — not a first aid certificate.

 OPEN / CLOSE

This Level 1 First Aid awareness course introduces important first aid concepts. It is general awareness only and does NOT make you a certified first aider.

Important — please read

Certification requires hands-on training

First aid certification (including the relevant NZQA unit standards) requires in-person training with a qualified instructor and assessment of practical skills like CPR. You cannot become certified online. Use this course to build awareness, then complete an accredited first aid course.

In any real emergency, call 111 first.
02

The basics of helping safely

Core principles that apply to any first aid.

 OPEN / CLOSE

Whatever the situation, a few principles always apply.

  • Check for danger before you approach — don't become a second casualty.
  • Call 111 for any serious injury or illness.
  • Stay calm and reassure the person.
  • Only do what you're trained and confident to do.
Your safety comes first. You can't help anyone if you're hurt too.
03

Building real skills

Why hands-on practice matters.

 OPEN / CLOSE

Skills like CPR, managing bleeding, and using an AED must be practised physically to be effective. Reading about them isn't enough.

  • CPR requires practised technique to be effective.
  • Managing serious bleeding needs hands-on training.
  • Using an AED is best learned with a trainer device.
  • Recognising serious conditions takes guided practice.

Book an accredited course

Find an accredited first aid provider and book an in-person course. That's how you gain real, certified skills.
